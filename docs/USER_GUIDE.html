🔒 1. Вхід у систему

Як увійти

  1. Відкрийте браузер та перейдіть за адресою системи (зазвичай http://localhost:5173)
  2. На сторінці входу введіть:
    • Ім'я користувача — логін, який вам надав адміністратор
    • Пароль — ваш пароль
  3. Натисніть кнопку «Увійти»
💡 Примітки
  • Сесія діє 8 годин, після чого система автоматично продовжить її (якщо ви активні)
  • Якщо сесія закінчилась — система перенаправить на сторінку входу
  • Для виходу натисніть на ваше ім'я у верхньому правому куті → «Вийти»

🚚 12. Переміщення ОЗ

Бокове меню → 📋 Операції → Переміщення

Оформлення внутрішнього переміщення основних засобів між підрозділами, місцезнаходженнями або матеріально відповідальними особами. Документ відповідає формі «Акт внутрішнього переміщення основних засобів» (затв. наказом Мінфіну від 13.09.2016 №818).

Як оформити переміщення

  1. Натисніть «+ Оформити переміщення»
  2. Заповніть шапку документа:
    • Номер документа — номер акту переміщення
    • Дата документа — дата переміщення
    • Звідки (місце) — місцезнаходження, з якого переміщується ОЗ
    • Куди (місце) — нове місцезнаходження (обов'язково)
    • Здав (МВО) — відповідальна особа, яка здає ОЗ
    • Прийняв (МВО) — відповідальна особа, яка приймає ОЗ (обов'язково)
    • Підстава — підстава переміщення (наказ, розпорядження тощо)
  3. Додайте об'єкти переміщення:
    • Натисніть «Додати ОЗ»
    • Оберіть основний засіб зі списку (пошук за інв. номером або назвою)
    • Балансова вартість підставляється автоматично
    • Вкажіть кількість та примітку (за потреби)
    • Можна додати кілька ОЗ в один акт
  4. Натисніть «Оформити»

📄 Документи

  • PDF — Акт внутрішнього переміщення основних засобів (форма згідно Наказу Мінфіну №818)
  • Excel — те саме у форматі Excel
  • Кнопки експорту доступні в рядку кожного запису

💡 Що відбувається автоматично

  • ✅ Оновлюється місцезнаходження та МВО для кожного ОЗ в акті
  • ✅ Створюється інформаційна бухгалтерська проводка (переміщення не змінює вартість ОЗ)
  • ✅ Записується в журнал аудиту
  • ✅ Надсилається сповіщення адміністраторам та бухгалтерам

💡 Примітки

  • При видаленні переміщення — місцезнаходження та МВО повертаються до попередніх значень
  • При редагуванні — система спочатку скасовує попереднє переміщення, потім застосовує нове
  • Один акт може містити кілька ОЗ, що переміщуються одночасно

🏢 15. Довідники

📍 Місцезнаходження

Перелік приміщень, будівель, територій.

👤 МВО

Особи, відповідальні за збереження ОЗ. Поля: ПІБ, Посада, ІПН, Місцезнаходження.

💼 Посади

Перелік посад для призначення МВО та користувачам. Посади відображаються у PDF-документах.

🏢 Організації

Дані організації для документів. Поля: Назва, Скорочена назва, ЄДРПОУ, Адреса, Керівник, Бухгалтер.

Дані організації автоматично підтягуються у всі PDF та Excel документи.

📄 16. Експорт документів (PDF / Excel)

Випадаюче меню (для звітів)

Кнопка «Завантажити ▼» з вибором: 📄 PDF або 📊 Excel.

Іконки в таблицях

📄 — PDF, 📊 — Excel.

  • PDF-документи відповідають типовим формам (ОЗ-1, ОЗ-3, ОЗ-4, ОЗ-6, Акт переміщення)
  • Excel-документи мають таку ж структуру
  • Документи автоматично підтягують дані організації

🏷️ 17. QR-коди

Реєстр ОЗ → кнопка QR-код

Система дозволяє генерувати QR-коди для кожного основного засобу.

Генерація для одного ОЗ

  1. Відкрийте Реєстр ОЗ або Картку ОЗ
  2. Натисніть іконку QR-код 🏷️
  3. Завантажте PNG або роздрукуйте для наклейки на обладнання

Масова генерація QR-кодів

  1. В Реєстрі ОЗ відмітьте потрібні об'єкти ☑️
  2. Натисніть «QR-коди (масово)»
  3. Система згенерує PDF з QR-кодами для всіх обраних ОЗ
💡 Поради
  • QR-коди зручно наклеювати на обладнання для ідентифікації при інвентаризації
  • Кожен QR-код містить інвентарний номер та назву об'єкта

📊 18. Масовий імпорт/експорт Excel

📤 Експорт у Excel

Реєстр ОЗ → «Завантажити ▼» → Excel

Вивантажує весь реєстр ОЗ у файл .xlsx з усіма колонками (інв. номер, назва, група, вартість, знос, статус, МВО, локація).

📥 Імпорт з Excel

Реєстр ОЗ → кнопка «📥 Імпорт Excel»

Підготовка файлу

  1. Завантажте шаблон натиснувши «Шаблон Excel»
  2. Заповніть рядки (кожен рядок = один ОЗ)
  3. Обов'язкові колонки: Інвентарний номер, Назва, Група ОЗ (1-16), Первісна вартість, Дата введення

Як імпортувати

  1. Натисніть «📥 Імпорт Excel»
  2. Оберіть підготовлений файл
  3. Перегляньте попередній результат
  4. Натисніть «Імпортувати»
✅ Автоматично
  • Валідація даних (унікальність інв. номерів, коректність дат)
  • Повідомлення про помилки у конкретних рядках
  • Запис у журнал аудиту

⏰ 25. Автоматичні задачі (Celery)

Система має фонові задачі, які виконуються автоматично за розкладом.

💡 Запуск
  • Windows: двічі клікніть start-celery.bat
  • Docker: запускаються автоматично (контейнери buh_celery та buh_celery_beat)

Розклад задач

ЗадачаКоли виконуєтьсяЩо робить
📧 Нагадування про амортизацію1 число місяця, 09:00Сповіщення бухгалтерам
⚠️ Перевірка зношеності ОЗЩопонеділка, 08:00Знаходить ОЗ зі зносом >90%
🔴 Повна амортизаціяЩопонеділка, 08:30Сповіщення про повністю замортизовані ОЗ
☁️ Автоматичний бекапЗа розкладомДамп БД → Google Drive
⚠️ Важливо
  • Без Celery сповіщення про амортизацію та знос не надсилаються
  • Автоматичне резервне копіювання не працює без Celery
  • Усі інші функції системи працюють без Celery

📋 26. Типові сценарії роботи

📥 Сценарій 1: Оприбуткування нового ОЗ
1
Реєстр ОЗ → «+ Додати ОЗ» → заповнити картку
2
Прихід → «+ Новий прихід» → оберіть ОЗ, тип, суму
3
Система автоматично створить проводку
4
Натисніть PDF/Excel для друку Акту ОЗ-1
📤 Сценарій 2: Списання ОЗ
1
Вибуття → «+ Нове вибуття»
2
Оберіть ОЗ, тип вибуття, причину
3
Система змінить статус на «Списаний»
4
Друкуйте Акт ОЗ-3 (або ОЗ-4 для транспорту)
📉 Сценарій 3: Щомісячне нарахування амортизації
1
Амортизація → оберіть рік/місяць
2
Натисніть «Нарахувати»
3
Система обчислить для всіх активних ОЗ
4
Завантажте Відомість амортизації
📦 Сценарій 4: Проведення інвентаризації
1
Інвентаризація → «+ Нова інвентаризація»
2
Заповніть реквізити, оберіть комісію
3
Натисніть «Заповнити активами»
4
Позначте знайдені ОЗ, вкажіть стан
5
Натисніть «Завершити»
6
Друкуйте Інвентаризаційний опис
💰 Сценарій 5: Переоцінка ОЗ
1
Переоцінки → «+ Нова переоцінка»
2
Оберіть ОЗ, вкажіть справедливу вартість
3
Система визначить тип (дооцінка/уцінка)
4
Перерахує первісну вартість та знос
🚚 Сценарій 6: Переміщення ОЗ
1
Переміщення → «+ Оформити переміщення»
2
Вкажіть номер, дату, звідки/куди, здав/прийняв
3
Додайте ОЗ: натисніть «Додати ОЗ» → оберіть зі списку
4
✅ Система оновить місцезнаходження та МВО
5
📄 Друкуйте Акт переміщення (PDF/Excel)
📊 Сценарій 7: Формування звітів за квартал
1
Оборотна відомість → оберіть період → «Завантажити»
2
Проводки → фільтр за датами → «Завантажити PDF/Excel»
3
Амортизація → перегляд зведеної відомості
💾 Сценарій 8: Налаштування резервного копіювання
1
Резервне копіювання → «Підключити Google Drive»
2
Авторизуйтесь через Google → оберіть папку
3
«Налаштувати розклад» → оберіть «Щодня»
4
Копії створюються автоматично щоденно
🏷️ Сценарій 9: Друк QR-кодів для обладнання
1
Реєстр ОЗ → оберіть потрібні ОЗ галочками
2
Натисніть «QR-коди (масово)»
3
Роздрукуйте PDF з QR-кодами
4
Наклейте на обладнання
📥 Сценарій 10: Масовий імпорт ОЗ з Excel
1
Реєстр ОЗ → «Шаблон Excel» (завантажте)
2
Заповніть дані ОЗ у шаблоні
3
«Імпорт Excel» → оберіть файл
4
Перегляньте попередній результат → «Імпортувати»
